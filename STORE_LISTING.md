## Privacy Practices

### Single Purpose
PrivacyBlitz has a single purpose: to help users discover and delete old accounts to improve their digital privacy.

### Host Permissions Justification
We request `<all_urls>` permission to:
- Scan browser history across all websites
- Detect account pages on any website
- Provide deletion links for any service

This is necessary for the core functionality of discovering accounts across the entire web.

### Data Handling
- **Data Collection**: None - all processing is local
- **Data Storage**: Local only using Chrome storage API
- **Data Transmission**: None - no external servers
- **Third-Party Services**: None
